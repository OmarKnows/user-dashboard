Greetings, and welcome to my project submission.

This application was made using Angular 18 and Angular Material, complemented by a selection of icons from Angular Icons.

It is important to highlight that global state management was intentionally omitted. Given the scope of this project, such an implementation would introduce unnecessary complexity without providing a meaningful benefit.

Additionally, the search functionality should not be client-sided. Performing searches on paginated data in the frontend would limit the search results to the current page, failing to provide the comprehensive results expected.
